# Mindset

An app for setting your mind on the things that matter most to you.

## Getting Started

### Requirements

1. Ruby
1. PostgreSQL (e.g. [Postgres.app][postgres-app])

### Setup

```sh
$ bin/bootstrap
```

### Testing

```sh
$ bin/rspec
```

### Running

```sh
$ bin/rails server
```

[postgres-app]: http://postgresapp.com
